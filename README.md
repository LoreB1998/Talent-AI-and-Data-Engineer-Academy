# 🎓 Talent AI & Data Engineer Academy

Repository dedicata al percorso formativo **Microsoft AI & Data Engineer Academy**, organizzato da [eForHum](https://www.eforhum.it/) in collaborazione con **Randstad** e **EOS Solutions**, con tecnologie e programmi formativi forniti da **Microsoft**.

## 📌 Obiettivo del corso

Formare figure professionali nel mondo dell'AI e dei dati attraverso una base tecnica comune che comprende programmazione in Python, cloud Azure, data engineering, intelligenza artificiale e l'ecosistema Microsoft Fabric.

## 🗓️ Durata

**11 maggio 2026 – 9 luglio 2026** · 248 ore · Full-time (4 giorni a settimana)

## 📜 Certificazioni previste

- **AZ-900** – Microsoft Azure Fundamentals
- **DP-900** – Microsoft Azure Data Fundamentals
- **AI-900** – Microsoft Azure AI Fundamentals
- **AI-102** – Designing and Implementing a Microsoft Azure AI Solution
